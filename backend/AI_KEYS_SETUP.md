# 🔑 Настройка API ключей для ИИ-функционала

## Обязательные ключи

### 1️⃣ GEMINI_API_KEY (Google Gemini)
**Для чего:** Генерация векторов (embeddings) для семантического поиска в Базе знаний

**Как получить:**
1. Перейти на https://aistudio.google.com/app/apikey
2. Войти в Google аккаунт
3. Нажать **"Create API Key"**
4. Скопировать ключ (формат: `AIzaSy...`)

**Стоимость:** 
- **БЕСПЛАТНО** до 1,500 запросов/день
- Далее: $0.0001 за 1,000 токенов

**Модель:** `text-embedding-004` (768 измерений)

---

### 2️⃣ GROQ_API_KEY (Groq Cloud)
**Для чего:** AI-ассистент (чат-бот) для методистов/учителей

**Как получить:**
1. Перейти на https://console.groq.com/keys
2. Зарегистрироваться/войти
3. Нажать **"Create API Key"**
4. Дать имя ключу и скопировать (формат: `gsk_...`)

**Стоимость:** 
- **БЕСПЛАТНО** до 14,400 запросов/день (огромный бесплатный лимит)
- Очень быстрая модель (до 500 токенов/сек)

**Модель:** `qwen/qwen3-32b` (очень умная китайская модель)

---

## Опциональные ключи (для синхронизации Google Drive)

### 3️⃣ GOOGLE_DRIVE_API_KEY
**Для чего:** Автоматическая загрузка учебных материалов из Google Drive в базу знаний

**Как получить:**
1. Перейти на https://console.developers.google.com/
2. Создать новый проект (или выбрать существующий)
3. Перейти в **"APIs & Services" → "Credentials"**
4. Нажать **"+ CREATE CREDENTIALS" → "API Key"**
5. **Включить Google Drive API:**
   - Перейти в **"APIs & Services" → "Library"**
   - Найти **"Google Drive API"**
   - Нажать **"Enable"**
6. Скопировать ключ

**Примечание:** Если не планируете синхронизировать документы из Google Drive - можно пропустить.

---

### 4️⃣ GOOGLE_DRIVE_FOLDER_ID
**Для чего:** ID папки с документами для синхронизации

**Как получить:**
1. Открыть нужную папку в Google Drive
2. Скопировать ID из URL:
   ```
   https://drive.google.com/drive/folders/1d9_a3NQ2hHioMJsaUJ53NIj-CS1rVeDd
                                            ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                            Это и есть FOLDER_ID
   ```
3. Папка должна быть **публичной** (доступ "Просмотр для всех")

## 📝 Настройка на сервере

### Метод 1: Через файл `.env` (локально)
```bash
cd /path/to/backend
nano .env
```

Добавить:
```env
GEMINI_API_KEY="AIzaSy..."
GROQ_API_KEY="gsk_..."
GOOGLE_DRIVE_API_KEY=""  # опционально
GOOGLE_DRIVE_FOLDER_ID=""  # опционально
```

### Метод 2: Через Docker (на продакшене)
Если проект в Docker контейнере:

```bash
# SSH на сервер
ssh root@185.217.131.26

# Найти контейнер
docker ps | grep backend

# Установить переменные через docker
docker exec <container_name> sh -c 'export GEMINI_API_KEY="AIzaSy..."'
```

**ИЛИ** через Coolify:
1. Зайти в Coolify → Выбрать проект
2. **Environment Variables**
3. Добавить ключи

## ✅ Проверка работоспособности

После установки ключей:

1. Перезапустить backend:
   ```bash
   docker restart <backend_container_name>
   ```

2. Проверить логи:
   ```bash
   docker logs <backend_container_name> | grep -E "GEMINI|GROQ|API"
   ```

3. Если ключи **НЕ** установлены, увидите:
   ```
   ⚠️  GEMINI_API_KEY не установлен. Embeddings и семантический поиск не будут работать.
   ⚠️  GROQ_API_KEY не установлен. AI-ассистент не будет работать.
   ```

4. Если всё ОК - предупреждений не будет.

## 🎯 Что будет работать без ключей?

| **Похожие статьи** | ❌ Не работает | ✅ Работает |
| **AI-ассистент (чат)** | ❌ Не работает | ✅ Работает |
| **Синхронизация Google Drive** | ❌ Не работает | ✅ Работает (если настроен) |
